## Overview
Network protocol analysis and configuration labs covering
DNS, HTTP, TCP and Cisco network infrastructure.

## Labs

### [Lab 1 — DNS & HTTP Analysis](./Lab1-DNS-HTTP-Wireshark/)
Packet capture and analysis of DNS and HTTP protocols using
Wireshark. Covers DNS resolution (nslookup, Type A/NS records),
HTTP GET interactions, conditional requests, multi-segment
downloads, parallel object fetching, and HTTP Basic Authentication.

### [Lab 2 — TCP Analysis](./Lab2-TCP-Wireshark/)
In-depth TCP analysis during a file transfer to gaia.cs.umass.edu.
Covers the 3-way handshake (SYN/SYN-ACK/ACK), sequence and
acknowledgment numbers, SACK, HTTP POST over TCP, RTT
measurement and EstimatedRTT calculation over 105 segments.

### [Lab 3 — Router Configuration, Static Routing & VLAN](./Lab3-Router-VLAN-Static-Routing/)
Network configuration using Cisco Packet Tracer. Covers router
setup via Console and Telnet, static routing with redundant paths
and administrative distance, VLAN creation and port assignment,
and inter-VLAN routing with Router on a Stick topology.
