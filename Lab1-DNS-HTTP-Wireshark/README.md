packet capture in a real network environment.

## Topics Covered

- DNS packet analysis (queries, responses, TTL)
- HTTP GET / Response interaction
- Conditional HTTP GET (If-Modified-Since)
- Large HTML document download (multi-segment TCP)
- Embedded objects (parallel downloads)
- HTTP Basic Authentication (Base64 encoding)

## Tools Used
- Wireshark
- nslookup
- Firefox
