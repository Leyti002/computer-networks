# Lab 3 — Router Configuration, Static Routing & VLAN

**Course:** INFO3221 — Computer Networks  
**Student:** Oumar Leyti Ndiaye | A00228981

## Overview
Network configuration and routing using Cisco Packet Tracer,
covering router setup, static routing with redundancy, and VLAN
segmentation with inter-VLAN routing.

## Topics Covered
- Router configuration via Console & Telnet
- Static IP configuration (Gigabit Ethernet interfaces)
- Static routing with administrative distance
- Multi-router topology with redundant paths
- VLAN creation and port assignment
- Router on a Stick (inter-VLAN routing)
- Security demonstration (IP spoofing without VLAN)

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI
